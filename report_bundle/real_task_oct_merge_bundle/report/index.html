OCT 三维体真实拼接任务

目标不是继续演示小 demo，而是处理真实工程约束：大体数据、48GB 显存预算、overlap 非固定先验、按 brick 流式输出，以及可复现实测 benchmark。

36GB

单个 `3000 x 1500 x 2000` float32 体约占用。

48GB

目标单卡显存预算，不能容纳两个完整 float32 体和输出。

~10%

overlap 只是先验范围中心，实际值由算法估计。

内存策略

输入体用 memmap 分块读取，融合阶段按输出 brick 拉取局部区域，不创建完整 stitched float32 体。

Overlap 估计

默认在 5% 到 20% 范围内低分辨搜索，使用 NCC 估计 axis=0 平移重叠。

真实 Benchmark

benchmark 会实际读取 stitched bricks 并组装切片，报告 disk_reads、mean_slice_ms、estimated_fps。

演示命令

先运行烟测验证包独立可用：

scripts\run_smoke_test.bat

替换真实数据后运行：

python scripts\run_real_pipeline.py --volume-a real_data\volume_a.npy --volume-b real_data\volume_b.npy --output-dir real_run
